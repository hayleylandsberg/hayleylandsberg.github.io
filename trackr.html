Trackr

A Job Search Tracker Application.

Provides users with an easy and fun way to organize their job search. The app was built using Django REST, React, and CSS.

Technology

PROCESS

I built this app for my Back End Capstone project while at Nashville Software School. I was given two weeks to build a useable application by myself. I started by creating a well thoughtout ERD, using draw.io to visually create it. Once the data structure was completed, I started wireframing the app and drawing out the separate components. I drew the wireframes on paper and created the prototype using Sketch. I used Github project board to manage my issues and stay on track with the project.

REACT & DJANGO REST

I built the web application using React for the client side and Django REST for the server side. I chose to use React because of its use of the virtual DOM. Any new view changes are first performed on the virtual DOM, which lives in memory and not on your screen. I

RESULTS

Once I completed all the components for my MVP, I let several people at Nashville Software School know that the app was available for use. Several people have reached out saying they cloned my app down from Github and used the app during their job searches. I still have several stretch goals I would like to complete before hosting it. I would like to add a Tasks section to each of the job cards, and allow users to view all of the tasks in one location. I'd also like to add metrics on how their job search is going, depending on dates the jobs are added and how many jobs are in each section.
